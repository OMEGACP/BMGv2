# Roster Building Files for Batman Miniature Game v2 from Knight Models.
The catalogue files were built using the New Recruit Editor.

Current state
===============
- all character traits, special attacks, effects and weapon traits from BMG Compendium version 1.3

Generic Henchmen
---------------
- all generic henchmen of BMG v2

Free Agents
---------------
- currently available free agents of BMG v2:
   - Copperhead
   - Ratcatcher

League of Assassins
---------------
- all equipment rules from the v2 rulebook
- all additional equipment rules from the v2 & v3 character cards (some rules migrated from v3 to v2)

Bane
---------------
- all models from the Bane faction of the Dark Knight Rises box (BMG v2), that can be used in the League of Assassins faction

# Extension
Contributions are welcome. Clone this repository and create a new branch with pattern "feature/*" or "bugfix/*".
It is recommended to only change a single faction in a single branch to ease pull requests.
The main branch is protected, so please allow for some review time.

# Disclaimer
There may be errors in the database. Create an issue or fix the error yourself in a new branch.
Rules migrated from v3 to v2 are subject to discussion. The following rules have been migrated:
- Falconry
- Shadows Command
- Lazarus Pit Owner
